Төслийн нэр: AI Digital Notebook Application
Энэхүү төсөл нь хэрэглэгчид тэмдэглэл хөтлөх боломж олгодог дижитал notebook аппликейшн бөгөөд AI технологи ашиглан автомат 
sticker дизайн үүсгэх болон notification (мэдэгдэл) илгээх функцуудтай.
Апп нь хэрэглэгчийн бүтээмжийг нэмэгдүүлэх, тэмдэглэлээ илүү ухаалаг, интерактив байдлаар удирдах зорилготой.

Ашигласан технологи, хэрэгслүүд
Frontend: React.js / Flutter 
Backend: Node.js + Express
Database: MongoDB
AI Integration: OpenAI API 
Version Control: Git & GitHub
Tools: Postman, VS Code, 

(Installation & Setup)

git clone (https://github.com/Anunaa02/Anu_AiNoteBook.git)
cd Anu_AiNoteBook

Backend ажиллуулах
cd ai-notebook-backend
npm install
npm start

Frontend ажиллуулах
cd ai_notebook_app
npm install
npm start

Environment тохиргоо (.env)
MONGO_URI=your_mongodb_url
OPENAI_API_KEY=your_api_key
JWT_SECRET=your_secret_key
