\chapter{Дүгнэлт}
\label{sec:conclusion}

Энэхүү судалгааны ажлаар Flutter дээрх харагдах хэсэг (frontend), Node.js/Express дээрх сервер талын хэсэг (backend), MongoDB өгөгдлийн сан болон OpenRouter API үйлчилгээг нэгтгэсэн ``AI Notebook'' системийг бодит түвшинд хэрэгжүүлж, баталгаажуулалтын урсгал, тэмдэглэлийн CRUD үйлдэл, хиймэл оюун ухаанаар стикер үүсгэх үндсэн боломжуудыг интеграцийн түвшинд амжилттай туршиж нотоллоо. Судалгааны үр дүнгээс үзэхэд модульчлагдсан архитектур, JWT-д суурилсан хамгаалалт, өгөгдлийн урсгал болон дарааллын диаграммын уялдаа нь системийн ажиллагааг ойлгомжтой, өргөтгөх боломжтой, инженерчлэлийн хувьд хэрэгжихүйц шийдэл болгож байгаа нь тогтоогдсон бөгөөд академик судалгааг бодит програм хангамжийн шийдэлтэй уялдуулсан практик ач холбогдолтой ажил болсныг харуулж байна.

 \section{Цаашдын төлөвлөгөө}
Цаашид ``AI Notebook'' системийг илүү ухаалаг болгох зорилгоор AI функцийг сайжруулж, тэмдэглэлд үндэслэн автомат ангилалт гаргах боломж нэмнэ. Мөн AI sticker generation-ийг хөгжүүлж, илүү олон стиль болон сонголттой болгоно. Хэрэглэгчийн мэдээллийн аюулгүй байдлыг сайжруулж, хоёр шатлалт нэвтрэлт болон нэмэлт хамгаалалтын систем нэвтрүүлэхээр төлөвлөж байна. Системийг web болон desktop хувилбарт өргөтгөж, олон төхөөрөмжөөс ашиглах боломжийг нэмэгдүүлснээр хэрэглэгчдийн өдөр тутмын тэмдэглэл хөтлөх үйл ажиллагааг илүү үр дүнтэй болгох боломжтой.
